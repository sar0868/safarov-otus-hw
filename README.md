## Старт игры. Спавн врагов

Цель:
В данном домашнем задании вы научитесь писать контроллеры и системы в Unity.


### Описание/Пошаговая инструкция выполнения домашнего задания:
Запрограммировать старт игры и спавн врагов:
- Спавн врага в одной из заданных точек
- Рандомный настраиваемый период спавна
- Контроллер в режиме паузы, если текущих врагов >= 10

Компетенции:
1. Программировать на C#
    - Реализовывать спавн врагов
